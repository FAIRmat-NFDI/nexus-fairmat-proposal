NXimage_set_em_ronchigram¶

Status:

base class, extends NXobject

Description:

Container for reporting a set of images related to a ronchigram.

Ronchigrams are specifically used in transmission electron microscopy to judge the settings for the aberration corrections and alignment.

Symbols:
